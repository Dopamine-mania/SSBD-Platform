### 4. 日历渲染算法

✅ **时间槽计算**:
```python
def time_to_row(time) -> int:
    hour = time.hour
    minute = time.minute
    row = hour * 2  # 每小时2个槽位
    if minute >= 30:
        row += 1
    return row
```
